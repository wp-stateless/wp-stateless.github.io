Features

We created this plugin for ourselves to make WordPress sites load faster and keep updating and improving it. It also has compatibility modes with popular WordPress plugins.

Images Safety

The Google Cloud Storage service typically stores at least three copies of each piece of the data, all in different places.

Increase speed on your site

Google Cloud Storage is geo-redundant, meaning your media is delivered by the closest server – reducing latency and improving page speed.

Work on multiple containers/servers

Scale your WordPress website across multiple servers without the need of synchronizing media files.

Clear a space

Media files are not stored locally in Stateless mode. That will free your server from the extra load and give you the ability to save your money from high recurring payments.

Native integration between GCS and WP.

Upload and serve your WordPress media with the WP-Stateless plugin

2 minutes to set up

WP-Statless Set up Assistant will navigate you through 3 simple steps

Improve WordPress Media storage

Once you uploaded the WP-Stateless plugin, you will be redirected to Set up Assistant which will navigate you through all the 3 steps

Login with the Google Account you want to be associated with your website and consent to the permissions request.

Set up Google Cloud Project and Bucket names. By default we are creating new project and bucket for you

That's all. Try to upload something. Any media file you upload to WordPress will now be uploaded to Google Cloud Storage and served to your users from Google servers!
